自家製酵母パン屋 A（bakery-A）

## 概要
SNSでのフォロワー数は多いものの、店舗での実売（完売）に結びつかないという課題を解決するための、「週末限定・予約専用」ランディングページ（LP）です。
単なる紹介サイトではなく、SNSの熱量を「確実な予約」へと変換させるためのクロージングツールとして設計・実装しました。

- ユーザーの目を引くスクロールアニメーション
- 直感的に操作できる予約フォームへの誘導ボタン

## 制作意図（課題解決アプローチ）
1. 「期待感」を最大化する導入（ヒーローセクション）
アプローチ: トップに「写真だけでは伝えきれない」という、あえて視覚情報を否定するキャッチコピーを配置。画面越しでは届かない「重厚感」や「香り」をユーザーに想像させ、スクロールを促すフックとして機能させています。

2. 「限定30本」の希少性を裏付けるストーリー
課題: なぜ完売しないのか → ユーザーが「いつでも買える」と思っている、または価値が伝わりきっていない。

解決策: 「48時間熟成」「職人の手作業」といったプロセスを可視化。なぜ30本が限界なのかという論理的な裏付けを行い、読了後に「予約してでも手に入れる価値がある」と確信させる構成にしました。

3. 日本語の美しさを引き出すデザイン
タイポグラフィ: 高級感と信頼感を醸成するため、明朝体（Zen Minchoなど）をベースにしたタイポグラフィを採用。余白を広めに取り、一文字ずつ丁寧に読ませる雑誌のような（エディトリアルな）レイアウトで、ブランドの品格を表現しました。

## 工夫した点
- シームレスな予約誘導: 各セクションの終わりに適切なタイミングでCTA（予約ボタン）を配置。「欲しい」と思った瞬間を逃さない導線設計を行っています。
- スライドショーの実装: SNSの「美味しそう」という空気感をそのまま伝えるため、自動再生のスライドショーを配置。視覚的な情報の鮮度を保ちます。
- 表示パフォーマンス: モバイル環境での閲覧を重視し、画像の最適化とシンプルなコード構成でストレスのない閲覧体験を提供。

## 学んだこと・今後の課題
- 学んだこと:
- ユーザーの心理的障壁（予約の手間、売り切れへの不安）を、デザインとライティングでどう取り除くかの重要性。
- デザインの「雰囲気」を崩さずに、操作性（アクセシビリティ）を確保するバランス。

- 今後の改善点:
- 予約状況を表示する動的なコンポーネントの実装（JavaScript）。
- 予約ボタンへのホバーエフェクトの強化など、より直感的なUIへの改善。
- 店舗情報のセクションにGoogleマップ等の詳細情報を拡充し、初見ユーザーの信頼度をさらに向上させる。
