自家製酵母パン屋 A（bakery-A）

## 概要
SNSで話題性は高いものの、当日完売に至らないという課題を抱えるパン屋さんのための、**週末限定食パン予約専用ランディングページ（LP）**です。
「SNSのフォロワーを確実に来店・購買へ繋げる」ことを目的とし、20代後半〜40代前半をターゲットに予約への導線設計とブランドの世界観を両立させたデザイン・実装を行いました。

## 実装内容
- セクション構成: コンセプト / 商品紹介 / 予約フロー / 店舗情報 / CTA（予約ボタン）
- レスポンシブ設計: スマートフォンでの閲覧・予約操作をメインに想定したモバイルファースト設計
- インタラクション:
- ユーザーの目を引くスクロールアニメーション
- 直感的に操作できる予約フォームへの誘導ボタン

---

## 制作意図（課題解決アプローチ）
- 1. ターゲットと目的の明確化
ターゲット: SNSでパン屋のアカウントをフォローしている「パン好き」な層。
課題: 「いいね」はつくが、わざわざ店舗に足を運ぶまでの「最後の一押し」が足りない。
解決策: 「週末限定」「予約制」という特別感を演出し、LP内で予約を完結させる仕組み（または予約フォームへの明確な誘導）を作ることで、購買行動を促進。

- 2. デザインのこだわり
パンを連想させる配色と、高級感を意識した雑誌のような（エディトリアルな）タイポグラフィを採用。
商品写真を大きく配置し、視覚的に「食べたい」と思わせるシズル感を重視。
SNSでの写真をフォトスライドとして配置し、視覚的にもアプローチする。

---

## 工夫した点
- CTA（行動喚起）の配置: ユーザーがスクロール中にいつでも予約を検討できるよう、視認性の高いCTAボタンを適切なタイミングで配置。
- 表示速度の意識: 画像の最適化を行い、モバイル環境でもストレスなく閲覧できるよう配慮。
